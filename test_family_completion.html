Test Instructions for Completed Family Animation

How to test the completed family animation:

  1. Open the game at http://localhost:8081
  2. Start a new game with any theme and settings
  3. Open browser developer tools (F12)
  4. In the console, run the following script to give yourself a complete family:

Test Script (paste in browser console):


// This script will give the current player all cards of the first family

// Get the game state from React DevTools or window object

What to look for:

  • ✅ Cards animate smoothly from hand to family count position
  • ✅ Animation duration is about 750ms (not too fast or slow)
  • ✅ Cards are removed from hand after animation completes
  • ✅ Toast notification appears saying "Famille complétée!"
  • ✅ Family count increases by 1
  • ✅ No visual glitches or state inconsistencies

Expected Behavior:

When a player completes a family (gets all 4 cards), the cards should animate from their hand position to the family count area, then after a delay, be removed from the hand and the family count should increase.
